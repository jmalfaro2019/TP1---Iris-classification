Resumen del Proyecto

Este proyecto implementa un análisis completo del dataset Iris, un benchmark clásico en Machine Learning. Se compararon múltiples algoritmos de clasificación supervisada para predecir especies de iris basándose en medidas morfológicas de sépalos y pétalos. El proyecto incluye análisis exploratorio, visualizaciones avanzadas y evaluación rigurosa de modelos.

Dataset Iris

Características del Dataset

El dataset Iris es posiblemente la base de datos más conocida en la literatura de reconocimiento de patrones. Contiene 3 clases de 50 instancias cada una, donde cada clase se refiere a un tipo de planta iris.

Características (Features)

  • Sepal Length: Longitud del sépalo (cm)
  • Sepal Width: Ancho del sépalo (cm)
  • Petal Length: Longitud del pétalo (cm)
  • Petal Width: Ancho del pétalo (cm)

Clases (Target)

  • Iris Setosa: 50 muestras
  • Iris Versicolour: 50 muestras
  • Iris Virginica: 50 muestras

Propiedad importante: La clase Iris Setosa es linealmente separable de las otras dos, mientras que Iris Versicolour e Iris Virginica no son linealmente separables entre sí.

Análisis de Resultados

El LDA demostró ser el modelo más efectivo con una precisión del 98% y baja variabilidad. Los modelos lineales (LDA, QDA) superaron a los no lineales en este dataset específico.

El Perceptron mostró un rendimiento inconsistente debido a la no separabilidad lineal completa del dataset.

Importancia de Características

El análisis del árbol de decisión reveló que petal length es la característica más importante (89.3%), seguida de petal width (8.8%).

Las características del sépalo mostraron menor poder discriminatorio para este problema de clasificación.

Hallazgos Clave

Separabilidad de Clases

La visualización con t-SNE y pairplots confirmó que Iris Setosa es claramente separable, mientras que Iris Versicolour e Iris Virginica presentan cierto solapamiento en el espacio de características.

Características Discriminatorias

Las medidas del pétalo (length y width) son significativamente más informativas que las del sépalo para distinguir entre especies, con petal length siendo la característica más importante.

Rendimiento de Modelos

Los modelos lineales (LDA, QDA) superaron a enfoques más complejos, sugiriendo que las relaciones lineales capturan adecuadamente la estructura subyacente del dataset Iris.

Conclusiones

Este proyecto demostró la aplicación efectiva de múltiples algoritmos de clasificación en el dataset Iris. Los resultados confirman que para problemas con estructura lineal subyacente, modelos simples como LDA pueden superar a enfoques más complejos.

Contribuciones Principales

  • Implementación comparativa de 5 algoritmos de ML
  • Análisis exhaustivo de importancia de características
  • Visualizaciones avanzadas para interpretación de resultados
  • Evaluación rigurosa con validación cruzada

Aprendizajes Clave

  • La simplicidad del modelo no siempre compromete el rendimiento
  • El análisis exploratorio es crucial para entender los datos
  • La validación cruzada proporciona estimaciones más robustas
  • La interpretabilidad del modelo es tan importante como la precisión
